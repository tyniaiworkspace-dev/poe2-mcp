## Installation

### PyPI (Recommended)
```bash
pip install poe2-mcp
```

### From Source
```bash
git clone https://github.com/HivemindOverlord/poe2-mcp.git
cd poe2-mcp
pip install -e .
```

### .mcpb Bundle
Download from [GitHub Releases](https://github.com/HivemindOverlord/poe2-mcp/releases/latest) for one-click Claude Desktop installation.
